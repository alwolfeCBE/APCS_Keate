Cipher Disk

Spies have used cipher disks for centuries to send coded messages to each other. One type of cipher disk has inner and outer disks that rotate to help create a cryptogram, a secret message. The disks set the letters to a code by setting A to another letter. All letters follow the same pattern. For example, if A is set to E, B would be set to F, C would be set to G, and so on. You can read more about the cipher disk in this document from the U.S. Department of Defense.
