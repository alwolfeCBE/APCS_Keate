Report Cards

Schools have different parameters for eligibility for extracurricular activities based on students' grades. Often times, eligibility has multiple dependencies, not just GPA. Suppose a school has the following terms of eligibility terms:
1. A student must be taking 4 or more classes.
2. A student must have a GPA of over 2.0.
3. A student must not have any F's.

In order for a student to be eligible for extracurricular activities, all of these terms must be true.

Assignment:
Write a program to find a person's GPA and eligibility status when given their grades. The program must print out a student's GPA and eligibility status along with a specific statement explaining ineligibility (if one is ineligible).

The following table shows some examples of different cases with the specific phrase that must be printed.

GPA: Number of classes F's Eligibility Statement:
4.0 5 No GPA: 4.0, Eligible
4.0 2 No GPA: 4.0, Ineligible, taking less than 4 classes.
3.0 5 Yes GPA: 3.0, Ineligible, GPA above 2.0 but has an F grade.
1.8 5 No GPA: 1.8, Ineligible, GPA below 2.0.
1.8 4 Yes GPA: 1.8, Ineligible, GPA below 2.0 and has an F grade.

1. Code the constructor with parameters. 2. Create the necessary getter methods to make sure the instance variables are accessible.

3. Write the method, eligibility, to return a String. The String must follow the table above - any different formatting will cause the Check button below to print Incorrect. Be aware that number of classes supersedes any of the other eligibility requirements.
Be aware that taking 4 or more classes supersedes any of the other eligibility requirements. In other words, if a student is only taking 3 classes and has a 1.8, eligibility should return the phrase:
GPA: 1.8, Ineligible, taking less than 4 classes.
4. Code the beginning of the client class. You will need to declare the gpa, number of classes, and whether there are F grades and initialize them to default values. Then, you should read the grades in one by one until a person enters an invalid grade (only A, B, C, D, and F) are valid grades. You should calculate the gpa, change hasF to true if an F is entered, and set numClasses to however many grades are entered.
Remember to use a loop to read in the grades and update your variables!
The Scanner class cannot read in chars, so read the user input in as Strings.
5. Create a ReportCards object with the values read in to test all the eligibility requirements by printing the eligibility statements. Your program will need to be run and tested multiple times. The following table holds test values and their expected results.
Grades: Eligibility Statement:
A, A, A, B, F GPA: 3.0, Ineligible, GPA above 2.0 but has an F grade.
A, A, A GPA: 4.0, Ineligible, taking less than 4 classes.
F, F, D, D GPA: 0.5, Ineligible, GPA below 2.0 and has an F grade.
C, B, D, D GPA: 1.75, Ineligible, GPA below 2.0.
A, B, C, A GPA: 3.25, Eligible

6. Make sure your name is commented out near the top of your source code.
